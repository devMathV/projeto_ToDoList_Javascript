# To Do List em Javascript

## Objetivo
O projeto teve como objetivo o exercício de conceitos aprendidos durante um curso de Javascript.

## Conceitos
Nesse projeto foi treinado alguns conteúdos como manipulação de elementos por meio do DOM, uso de funções, boas práticas de código.

## Como usar?
É possível adicionar tarefas colocando um título (obrigatório) e clicando no botão " + ".
As tarefas podem ser visualizadas como blocos, onde têm o título e dois botões, um para marcar como concluído e o outro para remover a tarefa.
- O botão de "concluir" torna a tarefa concluída, porém é possível reverter e tornar a tarefa como pendente clicando novamente nesse botão.
- O botão de "remover" remove a tarefa, sendo uma ação irreversível. Diferente do botão de "concluir" que ainda permite ver a tarefa.
